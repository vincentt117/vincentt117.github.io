Yeezy Vision

Yeezy Vision is a Chrome extension that lets you view the web from the eyes of the 21st century's greatest artist - Kanye West. Billions of people have been deprived of the life changing perspective that Mr. West has cultivated." A Google Chrome extension which embodies the persona of American celebrity Kanye West. The project was completed in 24 hours between February 21 - 22, 2016 at McHacks of McGill university. The extension was planned, developed, and tested by a team of four. The extension replaces keywords in articles with phrase associated with West and uses Microsoft Azure facial recognition technology to replace pictures of faces with that of West's.

Extension won best Google Chrome API Award at McHacks 2016 in McGill University

European Chess

European Chess game created via Java using Eclipse as the IDE. The project was the term project for the grade 11 introduction to computer science course at the Earl of March Secondary school. The game allows for 2-players using standardized chess rules with piece placement generated at the start of each new game. Extensive documentation was provided for the project and a journal which tracked project progression was maintained throughout development

Developed in under 24 hours, Franklin allowed investment firms to track the progress of start-ups they have invested in through custom KPI questions that formulate visual data representations. DocTalk

A "virtual clinic" Android app developed during Concordia University's ConU Hacks II via Android Studios, written with .xml and Java. The application has two user types: standard and doctors. Anyone can have a standard account, these accounts allows one to send out medical requests, stating area of discomfort, level of discomfort, additional comments, etc., Doctor accounts can view the incoming requests as a queue and provide their diagnosis, send these out, and requesting users will receive them. Application was very well received at the event and is currently being considered by the Children’s Hospital of Eastern Ontario (CHEO) for internal use

Spicy AI

A web-service built on Node incorporating Nuance's MIX API, IBM's Watson Tone Analysis API, and embodiment of giphy searches. The services takes a spoken phrase and converts it to string via MIX API, sends it to Tone Analysis for evaluation of emotional value (a score for each of five emotions: joy, fear, anger, disgust, and sadness), the emotion of greatest magnitude is then used to search through giphy, where relevant .GIF loops are presented to the user on the web-app.
